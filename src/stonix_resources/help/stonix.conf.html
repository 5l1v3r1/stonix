stonix.conf

The stonix.conf configuration file holds all of the custom configuration changes made to STONIX. This consists of configuration items (CIs) and user comments. By default, the stonix.conf file is populated with each rule's name and help text, but not with default CI values. To create a fully-populated stonix.conf file, execute stonix -cP at the command line (a minimally-populated stonix.conf can be generated by executing stonix -cp).

The format for setting a rule's CI(s) is as follows:

[RuleName]
CINAME = civalue
UCCINAME = User comment text goes here, and it can be whatever you want! No quotes needed.
CIs can take several different data types, but the most common is boolean (True/False). These CIs appear as checkboxes in the GUI. To set the value of a boolean CI, simply set CI = True or CI = False.
