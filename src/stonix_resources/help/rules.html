Introduction to Rules

A stonix rule is a logical collection of configuration items. A configuration item is a unit that maps to a configuration in a security standard. For example, the BlockSystemAccounts rule maps to CIS and NSA configuration guidance, while PreventXListen maps to only one piece of configuration guidance.

Some rules have multiple configuration items that may or may not be configurable. For instance, DisableCloudServices has two configuration items, which both are configurable.
