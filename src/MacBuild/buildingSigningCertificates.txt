# Setting up certificates to sign stonix4mac builds

1. Get a USB with the team's Developer ID certificates from Roy Nielson.

2. Mount the USB and open keychin utility on your build environment.

3. To add certificates to your keychain, drag them from the usb to the keychain utility. If adding the certificate requires a password, ask Roy.
The certificates need to be added in the following order:
Apple_WWDRCA
DeveloperIDCA
DeveloperID_application
DeveloperID_installer
Application
Install

#####
#
# After this, you should be able to sign stonix builds.
#
# To build stonix, refer to buildingStonix.txt
